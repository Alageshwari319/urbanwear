# UI-UX-EXPERIMENT2

## AIM:
To design a modern and user-friendly fashion e-commerce mobile interface named UrbanWear using Canva by applying Gestalt principles, UI components, usability principles, and interaction design techniques for creating an effective login page and home page layout.

## ALGORITHM:
### Step 1: Requirement Analysis
Identify the purpose of the application.
Choose the website category as a fashion e-commerce platform.
Define target users and required screens.

### Step 2: Brand Creation
Create the brand name “UrbanWear”.
Design a logo using Canva with fashion-inspired styling and typography.
Select suitable brand colors and fonts.

### Step 3: Login Page Design
Create a mobile login layout in Canva.
Add background illustration and branding elements.
Insert username and password input fields.
Add login and sign-up buttons.
Apply Gestalt principles such as:
Figure-Ground
Similarity
Proximity

### Step 4: Home Page Design
Design the mobile home page layout.
Add navigation bar and search functionality.
Create fashion category cards with images.
Arrange product sections using grid layout.
Apply Gestalt principles such as:
Continuity
Similarity
Figure-Ground
Proximity

### Step 5: UI Components Integration
Include search bars, buttons, icons, and image cards.
Use rounded corners and consistent spacing.
Maintain visual hierarchy and responsive layout.

### Step 6: Usability and Interaction Design
Ensure easy navigation and accessibility.
Use readable typography and proper contrast.
Add interactive elements such as buttons and category selection.

### Step 7: Finalization
Review the overall layout consistency.
Export the final screens from Canva.
Prepare documentation with screenshots and explanations.
